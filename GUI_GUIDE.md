# 🎮 大富豪 GUI - ユーザーガイド

グラフィカルインターフェース（Pygame）を使った、人間 vs AI の大富豪対戦システムです。

---

## インストール

### 前提条件

- Python 3.8以上
- 仮想環境が有効

### 必要なパッケージ

```bash
# メイン依存関係（既にインストール済みの場合がほとんど）
pip install gymnasium stable-baselines3 numpy

# GUI 用追加パッケージ
pip install pygame
```

---

## 起動方法

### 基本的な起動

```bash
python gui.py
```

**初回実行時:**

- モデルが自動的にロードされます（`models/daifugo_ppo.zip`）
- 訓練済みモデルが見つからない場合、ランダムAIで対戦します
- Pygame ウィンドウが起動します（1200x800px）
- 日本語フォント（msゴシック、Hiragino Sans）が自動選択されます
- フォントが見つからない場合、デフォルトフォントで表示されます

### ウィンドウが死活してしまった場合

```bash
# キャッシュをクリアして再起動
python gui.py
```

## 操作方法

### ターン中の操作

- **クリック**: 下部の手札からカードを選択
  - 選択されたカードは金色でハイライトされます
  - グループやストレート（複数枚）の場合も自動的に判定されます

- **Space キー**: 選択したカードを出す（確定）
  - 複数枚を出す場合のパターン検出は自動で行われます
  - 非合法な手は自動的に拒否されます

- **P キー**: PASS（現在のターンをスキップ）
  - 場に出ているカードに勝てない場合に使用
  - パス後は AI のターンになります

- **ESC キー**: 選択を解除
  - 選択中のカードをクリアします
  - 気を変えて別のカードを選べます

- **← → 矢印キー**: 手札をスクロール 🆕
  - 左矢印キー: 手札を左へスクロール
  - 右矢印キー: 手札を右へスクロール
  - 27 枚すべての手札を確認できます

### ゲーム中の操作

- **ウィンドウの×ボタン**: ゲーム終了

## ゲームルール

### 基本ルール（大富豪）

1. **カードの強さ**
   - 通常時: 3 < 4 < 5 < ... < J < Q < K < A < 2 < 🃏
   - メディリア(赤) > ジョーカー(黒)

2. **出し方のパターン**
   - **シングル**: 1枚出す
   - **グループ**: 同じランクのカード 2〜4 枚
   - **ストレート**: 同じスートで連続するカード 3 枚以上
     - 例: ♠3-4-5, ♥10-J-Q

3. **特殊ルール**
   - **革命**: グループ（同ランク 4 枚）を出すとカードの強さが反転
     - 通常時: 2 > A, 革命時: 3 > 2
   - **8 切り**: 8 のカードを出すと場がクリアされ、同じプレイヤーが続行
   - **パス**: 場に勝つ手がない場合にパス（AI も自動でパス判定）

### 勝利条件

- 手札が最初に 0 枚になったプレイヤーが勝ち
- ゲーム終了時に画面に「勝者: あなた」または「勝者: AI」が表示されます

## よくある質問（FAQ）

### Q1: ゲームが反応しない

**A:** 以下を試してください:

1. AI のターン中です（画面に「AI のターン」と表示）
   → 1〜2 秒待ちます

2. ウィンドウをクリックしてアクティブにします

3. 最悪の場合、Ctrl+C で終了し、`python gui.py` で再起動

### Q1b: フォントが文字化けしている

**A:** システムフォントが見つからない場合です：

1. **Windows**: msゴシック、MS P ゴシック を確認
2. **macOS**: Hiragino Sans を確認
3. **Linux**: 以下を実行:
   ```bash
   # フォント確認
   fc-list | grep -i gothic
   # または
   fc-list | grep -i sans-serif
   ```
4. GUI が自動的にシステムフォントを選択します
5. それでもダメな場合は、gui.py の font 設定を手動編集

### Q2: カードが選択できない

**A:** 以下の理由が考えられます:

- １）AI のターン中 → AI の行動完了を待つ
- ２）非合法な手 → 「合法手」を確認して別のカードを選ぶ
- ３）ゲーム終了後 → 新しくゲームを開始してください

### Q3: ゲーム画面の文字が小さい/大きい

**A:** `gui.py` の以下の行を調整してください:

```python
self.font_large = pygame.font.Font(None, 36)   # 大きい文字
self.font_medium = pygame.font.Font(None, 24)  # 中くらい
self.font_small = pygame.font.Font(None, 18)   # 小さい文字
```

数字を大きくすると文字も大きくなります。

### Q4: モデルをプリロード設定できるか

**A:** `main()` の以下の部分を編集:

```python
# ランダム AI で対戦したい場合
gui = DaifugoGUI(model=None)  # model=None でランダム

# または明示的にモデルをロード
from train import DaifugoTrainer
trainer = DaifugoTrainer()
trainer.load_model("models/daifugo_ppo.zip")
gui = DaifugoGUI(model=trainer.model)
```

### Q5: 画面解像度を変えたい

**A:** `DaifugoGUI.__init__()` の解像度を変更:

```python
# デフォルト: 1200x800
gui = DaifugoGUI(screen_width=1600, screen_height=1000)

# または
gui = DaifugoGUI(screen_width=800, screen_height=600)  # 小さめ
```

## トラブルシューティング

### Pygame がインストールできない

```bash
# 代わりにこれを試す
pip install pygame --upgrade

# または conda ユーザーの場合
conda install pygame
```

### モデルが見つからないエラー

```
⚠️  モデルが見つかりません。ランダム AI で対戦します。
```

→ 以下を実行してモデルを訓練:

```bash
python play_game.py
```

### ウィンドウが起動しない

Linux/Mac の場合、環境変数設定が必要な場合があります:

```bash
export SDL_VIDEODRIVER=windowed
python gui.py
```

## パフォーマンスのコツ

1. **AI の思考を高速化**
   - 訓練済みモデルを使用する（ランダム選択より高速）
2. **フレームレート調整**
   - `gui.py` の `self.clock.tick(30)` を変更
   - 30 → 60 でより滑らか（CPU 使用率↑）
   - 30 → 15 でより低負荷（反応性↓）

3. **メモリ許可範囲**
   - GUI 起動時: 約 150MB
   - ゲーム開始後: 約 200MB
